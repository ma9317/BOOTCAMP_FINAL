# BOOTCAMP_FINAL
**Predicting 30-Day Hospital Readmission Risk Among Patients With Diabetes**


This project developed a predictive model to identify diabetic hospital patients at risk of being readmitted within 30 days of discharge. The analysis used the Diabetes 130-US Hospitals for Years 1999–2008 dataset (from UC Irvine: https://archive.ics.uci.edu/dataset/296/diabetes+130-us+hospitals+for+years+1999-2008), which contains inpatient encounter data from 130 U.S. hospitals and integrated delivery networks. Each record represents a hospital encounter for a patient diagnosed with diabetes, including demographic information, admission details, diagnoses, laboratory testing, medication use, and readmission outcome. The original dataset was designed around the goal of predicting early readmission within 30 days. The main outcome variable was readmitted, which was converted into a binary target: patients readmitted in fewer than 30 days were coded as 1, while patients readmitted after 30 days or not readmitted were coded as 0. This framing allowed the project to focus specifically on early readmission risk, which is clinically important because 30-day readmission is commonly used as a healthcare quality.
Data preparation included replacing missing-value markers, removing identifier columns, dropping variables with excessive missingness, cleaning invalid categories, and transforming diagnosis codes into broader clinical groups. We performed additional feature engineering by converting age ranges into approximate numeric midpoint values and creating medication-related features such as the number of diabetes medications used and the number of medication changes during the encounter.
Our exploratory data analysis showed that 30-day readmissions were relatively uncommon compared with non-readmissions, creating an imbalanced classification problem. Visual analysis examined relationships between readmission status and age, time in hospital, number of medications, number of procedures, number of diagnoses, A1C results, insulin use, medication changes, and discharge-related variables. These patterns helped guide feature selection and model interpretation.
Several supervised learning models were developed and compared, including logistic regression, decision tree, and random forest models. Logistic regression served as the primary regression-based baseline because it is interpretable and appropriate for a binary outcome. Tree-based models were included as comparison models because they can capture nonlinear relationships and interactions between patient, hospital, diagnosis, and medication features. Prior research on this dataset has also found that medication features and tree-based models can be useful for readmission prediction.
Model performance was evaluated using accuracy, precision, recall, F1-score, confusion matrices, and ROC-AUC (performance metric for binary classification problems, plotting True Positive Rate (TPR) against False Positive Rate (FPR) at various thresholds). Because the positive class represented a smaller share of the dataset, accuracy alone was not sufficient. Recall and F1-score were emphasized because the practical goal is to identify patients who may need additional discharge planning or follow-up support.
Overall, the project found that patient readmission risk can be modeled using routinely collected hospital encounter data, although prediction remains challenging because early readmission is influenced by many clinical and social factors not fully captured in the dataset. The final model could be used as a decision-support tool to help flag higher-risk diabetic patients for interventions such as medication reconciliation, discharge education, care coordination, and early outpatient follow-up. Future work could improve the analysis by tuning model thresholds, testing additional algorithms such as gradient boosting, adding model interpretability methods, and evaluating fairness across demographic subgroups.
